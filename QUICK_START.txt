TROUBLESHOOTING:
================================================================================

Port Already in Use?
   Error: "Address already in use"
   Solution: Kill existing process or wait 30 seconds before restarting

Backend Won't Start?
   Error: "Python not found" or "No module named flask"
   Solution: Install Python dependencies
   Command: pip install -r backend/requirements.txt

Frontend Won't Start?
   Error: "npm not found" or "Module not found"
   Solution: Install npm packages
   Command: cd frontend && npm install

Browser Won't Open?
   Solution: Manually open http://localhost:3000

No Threats Showing?
   Normal: Wait 30 seconds for first threat detection
   Check: Backend console window for "ðŸš¨" messages
